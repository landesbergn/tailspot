Branch feat/leaderboard-trophies · dynamic-leaderboards PR3 · 2026-07-11

The three trophies

  • Top Flight (visible) — win a weekly leaderboard (weeklyWins ≥ 1). Laurel-and-star hex.
  • Dynasty (SECRET, masked ??? until earned) — win 3 weekly boards (weeklyWins ≥ 3). Stacked-crowns hex.
  • Chart Topper (visible) — ever hold #1 on the all-time board (everToppedAllTime). Summit-flag hex.

These are server-truth trophies: the backend alone decides wins (UTC Monday crowning, shared crowns count each sharer, no winner floor). The app reads the facts from the last leaderboard fetch, cached in LeaderboardStandingCache — it never infers a win locally. Offline the cache persists; a fresh install with no server contact shows all three locked.
