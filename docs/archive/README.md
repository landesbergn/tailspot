# docs/archive

Historical material, kept for the record and moved out of the active tree in
the 2026-07-12 pre-v1 cleanup. Nothing here describes current behavior — the
live docs are `PLAN.md`, `STRATEGY.md`, `CHANGELOG.md`, `CLAUDE.md`,
`docs/plans/` (in-flight plans only), `docs/runbooks/`, `docs/legal/`, and
`docs/testflight-handoff.md`.

- `superpowers/` — plans/specs/research from the retired superpowers workflow
  (May–June 2026). Several specs are still cited as provenance from source
  comments (pinhole projection, activity rarity, visual identity).
- `plans/` — dated plans for work that has shipped (see CHANGELOG for each).
- `reviews/` — one-time HTML review artifacts for shipped rounds.
- `brainstorms/` — dated brainstorm docs that fed shipped work.
- `backend-handoff.md` — the original backend provisioning runbook (the
  backend has been live at `api.tailspot.app` since 2026-06).
