Branch hig-compliance-fixes. Every image is before (left) | after (right) at iPhone-16 points, dark, default text size — except the three "scaling proof" shots, which are default (left) | accessibility-2 text size (right) on the new build. Full test suite green (957+ tests); reveal cards, settled cards, share card, trophy lists, and leaderboard windows are pixel-identical before→after.

What changed (and what deliberately didn't)

  • Dynamic Type everywhere prose/list content lives — Brand tokens ride text styles; mono(relativeTo:) anchors scrollable readouts; AR HUD + card artboards stay fixed by design.
  • Bold Text — B612 Mono maps to its Bold face when the system setting is on; system tokens adapt natively.
  • VoiceOver — AR planes are pinnable elements; the reveal announces the result; split-flap names read as words; rows/cards/tiles read as one sentence; decorative symbols hidden. No visual change.
  • 44 pt touch targets — 17 controls, almost all via invisible hit-area insets. Visible-size changes only where flagged below.
  • Reduce Motion — empty-tap ripple, segmented pills, completion ring now gate (the reveals already did).
  • Launch screen — now paints brand navy (was a white flash on light-mode phones). Not snapshotable; verify on device.
  • Deleted — orphaned CardReveal.swift (only its own #Preview constructed it).

On-device checklist (things snapshots can't show)

  • Launch the app with the phone in light mode — launch flash should now be navy, not white.
  • Settings → Accessibility → Larger Text → crank it: Settings/Profile/Hangar/rows should grow; the AR HUD and reveal cards should NOT.
  • Settings → Accessibility → Bold Text: mono readouts should switch to the B612 Bold face.
  • VoiceOver on the AR view: swipe through planes (nearest first), double-tap to pin, catch one — the reveal should announce “Caught …, …, … points.”
  • The Hangar sheet's new close pill — taste check the quiet chrome.
  • Capture bar sits ~6 pt higher (now clears the home-indicator zone) — feel check.
  • Map chips: LEGENDARY scrolls into view; tap targets should feel easy now.
